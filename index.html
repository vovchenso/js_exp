About Program

The program written in JavaScript. It used HTML5 canvas element for all drawings. Top area displaying some information from canvas area.

How To Use

To see the result You have to add 3 points to the canvas area. In order to add a point to the canvas you need to click somewhere inside canvas area. By clicking it will draw red circle. Then the program will calculate the coordinates for 4th point and draw the parallelogram and the circle with the same area as parallelogram.

Info about coordinates of the points and area you can see above.

Now you can start to drag red point by clicking on red point and moving it. The info above will be updated accordingly.

You can clear area and start from beginning by pressing button "Reset".
